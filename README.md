# 🎓 Student Productivity Hub

A modern and responsive **Student Productivity Hub** developed as part of the **ApexPlanet Software Pvt. Ltd. Web Development Internship**. This project is designed to help students organize their daily academic activities through an intuitive interface with useful productivity tools such as a Weather App, Todo List, responsive design, and interactive UI components.

---

## 📌 Project Overview

The Student Productivity Hub is a multi-page web application built using HTML, CSS, and JavaScript. It provides students with a centralized platform to manage their daily tasks, check weather updates, and enhance their productivity through a clean, responsive, and interactive user interface.

This project demonstrates front-end web development skills including responsive layouts, JavaScript interactivity, API integration, local storage, performance optimization, and deployment.
