## 📖 Overview
Student Productivity Hub is a fully responsive, modern web application designed to eliminate the chaos of academic life. It provides a streamlined, focused environment for learning by combining powerful productivity tools into a single, cohesive dashboard.

## ✨ Features
- **Responsive Design**: Flawless experience across mobile, tablet, and desktop devices.
- **Dark/Light Mode**: User preference saved via `localStorage` for a comfortable viewing experience.
- **Weather App**: Real-time weather data integration via OpenWeatherMap API with fallback mock data.
- **Todo List App**: Complete task management (Add, Edit, Delete, Filter) with data persistence in `localStorage`.
- **Interactive UI**: Includes an image carousel, smooth scrolling, modal popups, and animated counters.
- **Form Validation**: Real-time client-side validation for seamless contact form submissions.
- **SEO Optimized**: Fully integrated Meta tags, Open Graph tags, and lazy-loading images for peak performance.

## 💻 Tech Stack
- **HTML5**: Semantic and accessible markup.
- **CSS3**: BEM-inspired naming conventions, CSS Grid/Flexbox, and modular architecture.
- **JavaScript (ES6+)**: Vanilla JS for all logic, asynchronous API fetching (`fetch`), and DOM manipulation.
